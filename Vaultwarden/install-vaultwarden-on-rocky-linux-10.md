# How to Install Vaultwarden on Rocky Linux 10

Vaultwarden is a lightweight, self-hosted password manager compatible with Bitwarden clients, designed for secure credential storage without heavy resource usage. This guide introduces how to deploy Vaultwarden on Rocky Linux 10 for a private password management setup.
