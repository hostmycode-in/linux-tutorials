# How to Install Vaultwarden on AlmaLinux 10

Vaultwarden is a lightweight, self-hosted password manager compatible with Bitwarden clients, designed for secure credential storage without heavy resource usage. This guide introduces how to deploy Vaultwarden on AlmaLinux 10 for a private password management setup.
